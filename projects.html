Musique

Musique is a mobile app designed to play music on Android smartphones with minimum SDK Version API level 21 (Lollipop 5.0) and will run smoothly up until API level 28 (Android 9).
This project was an individual mid-term project with only 3 x 24 hours to finish.

Type Individual project
Role Full-stack developer
IDE Android Studio
min SDK API 21 (Lollipop 5.0)
Language Java

Preview of the app's features below

Login page
The username and password were predetermined and intregrated to the app. User will only be able to login using correct credentials.

Main page
The app will ask for permission to access phone storage upon first time running the app. Once granted, access will be saved and the app will continue to have access to the phone storage.

Musique will automatically reads all audio files saved in the device. The files' name and location will be displayed on the main page.
To select desired song, user can simply scroll and find the song or use the search bar above.

Side menu
Tap the three vertical dots on top right to access side menu. User can choose to go to My Profile, sort the list alphabetically, or to logout from the app.

Now playing
Selected song will be shown here. Press the play button to play the music and slide the slider timestamp left or right to jump backward or forward.

Profile page
Short profile of the app's creator (yours truly) and all references used.

Presetan

Presetan is a photo editing mobile app that will run smoothly on Android API level 21 (Lollipop 5.0) up until API level 28 (Android 9).
This one was a group project with 4 people in it including myself. We were told to make any kind of Android app that met some requirements including internal and external storage and phone sensors usage.
Here I worked as the front-end developer as well as solo artist who drew all the assets

Type Group project (4)
Role Front-end developer
IDE Android Studio
min SDK API 21 (Lollipop 5.0)
Language Java

Preview of the app's features below

Permission grant
Upon first time installing and running the app, it will ask for a permission to access your phone's storage. Press allow to grant access.

Main page
Here is what the main page should look like. You may choose where to get the photo to edit.

Choose from device
If you've chosen to choose from device, the app will ask which gallery viewer app you would like to use to access your gallery.

Select a photo from your gallery to proceed.

Edit page
You will be directed to edit page. Here you can choose from existing filters.
Click save on the top right corner to save your edit or click open to choose another photo from your gallery.

Let's try taking our own photo.
Go back to the main page and choose capture new image.

Presetan will open the camera app on your device (you should choose which one if you have more than one), snap a photo, then you may proceed to edit just like before.

Side menu
Go back to the main page and click on the vertical dots on top right corner to access side menu.
From here you can access your profile.

By default, user will be detected as guest. You may choose to register or sign in if you already have an account.
Please consider to sign up to unlock the app's full potential.

Register
Here is how the register page looks like. Fill in your email and password to sign up.

Login
This is the login page, use your registered credentials to login.

Once you're logged in, the side menu will add an option to sign out.

Access the profile once again and you'll see your new status as VIP. Congratulations.

That's not all, try editing photo once again as VIP user and you'll unlock new edit section beside filters where you can also edit the brightness and constraint of your photo.

thesosmed

thesosmed is a website based social media where users can edit their own unique mini profile, post updates, and leave comment for other users to see.
This was a mid term group project, my role in the team was as UI / UX designer. It's my job to initialy design how our website should look like and where every elements should go in order to help users surf on our website comfortably without getting confused of what to do.

You can try our website here. My partner rehosted the app just so you can experience our work firsthand.

Type Group project (6)
Role UI / UX designer
Language php

Preview of the website's features below

Login page
Where user logs in using their existing account or sign up if they have no account yet.

Sign up page
Here user can sign up for a new account by filling the form then click the sign up button.

After succesfully creating a new profile, user will be able to login right away using their signed up credentials.

Default profile
This is how every new user's profile looks like, with default NPC picture and basic information. It can be updated by clicking the edit profile button.

Edit profile
User can edit their profile information in this page. Click save changes to proceed with the changes or cancel to go back without making any change.

Your page
On your own page, you will be able to see your profile information on the left and it can edited by clicking edit profile.
The middle section is for updating your status. As for now, it's still limited to text only but hey you can comment on your own post so you won't feel as lonely when nobody does :D
All of the other users will be shown in the list on the right side, you can click on their name to visit their profile.

Other's page
When visiting other's page, their profile information will be on the left, while other users will still be on the right (it's just cropped in the screenshot).
All of the user's posts will be shown in the middle and you can leave a comment to their posts.
